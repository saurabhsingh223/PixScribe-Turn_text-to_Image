## Quick Start Guide for PixScribe

### Step 1: Install Dependencies (First Time Only)
```bash
npm install
```

### Step 2: Run the Application

**Easiest Way:**
```powershell
.\start.ps1
```

This will:
- Start the proxy server (Port 3001)
- Start the frontend (Port 5173)
- Open Chrome automatically

**You'll see 2 PowerShell windows - Keep them both open!**

### Step 3: Use the App

1. Chrome opens to `http://localhost:5173`
2. Click **"Generate"** in navigation
3. Enter a text description
4. Click **"Generate"** button
5. Wait 5-10 seconds
6. Your image appears!

## That's It!

- No API keys needed
- No signup required
- Completely free
- Generate unlimited images
